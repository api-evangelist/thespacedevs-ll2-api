# TheSpaceDevs LL2 API (thespacedevs-ll2-api)

TheSpaceDevs LL2 API is the Launch Library 2, a comprehensive space launch and events database delivering data on rocket launches, space events, astronauts, spacecraft, space stations, and crewed spaceflight missions worldwide. Provides real-time updates on launch schedules, mission timelines, and spacecraft operations.

## APIs

### TheSpaceDevs Launch Library 2 API

The Launch Library 2 API is the official successor of the popular Launch Library API. It provides comprehensive data on rocket launches, space events, crewed spaceflight, astronauts, spacecraft, space stations, and celestial bodies. The API is maintained by TheSpaceDevs community and delivers real-time and historical data on global space missions.

- **Human URL:** [https://thespacedevs.com/llapi](https://thespacedevs.com/llapi)
- **Base URL:** `https://ll.thespacedevs.com`
